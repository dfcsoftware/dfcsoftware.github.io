Notes

Published: 2025-10-14 10:20
Author: Don Cohoon

My goto note taker and also web bookmarker/saver.

It has all the features I need:

  1. Runs on all my devices
  2. Centrialized storage and backup
  3. Data sharing
  4. Works offline
  5. Private because it’s self-hosted
  6. Supports html, pictures, sound, video, attachments like pdf.
  7. Standard filesystem directory format, no special software needed to view.

My technique is ‘foldering’.

  1. Create a new E-Mail using any E-Mail client that’s handy
  2. Make it text or html
  3. Paste a web page, a URL, maybe type some thoughts into the body
  4. Add a meaningful subject
  5. Close, and save as Draft

Reasons I keep using it

I can move a note to a folder of my choice. Never have to hit ‘Send’.

Editing is just a matter of dragging it back to the Draft folder.

It comes with built-in full text search.

This can be shared with someone else by just hitting Send.

Self Hosted [Optional]

Because I host my own E-Mail Server (dovecot)[1] it is private by default, and these are in Maildir[2] format which is one file per message, so backup, restore and transfer are easy[3].

The Maildir folder is also on RAID storage, so hardware failures are easy to recover from.

If this is only for note gathering on local networks, I just use IMAP (port 143). For internet facing I use IMAPS (port 993) using ssl_cert (file /etc/dovecot/conf.d/10-ssl.conf).

In some configurations I will skip SMTP (port 25) outgoing mail if I don’t need it. A whole lot simpler without SPF, DKIM and DMARC, unless you have a ‘satellite’ system where all mail is sent to another local machine; an e-mail server.

Ubuntu has a nice setup wizard for SMTP using postfix:

 dpkg-reconfigure postfix

There are more details at E-Mail Quickstart[1] below.
